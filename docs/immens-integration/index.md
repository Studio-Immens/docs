# 🔗 Immens Integration

**Tipo**: FREE
**Plugin**: immens-integration

Plugin core di integrazione per tutti i plugin Studio Immens. Fornisce funzionalita' condivise come AI providers, webhook system, notification system, e licenze.

## Caratteristiche

- Gestione AI providers (OpenAI, Anthropic, Google, Ollama)
- Webhook system per automazioni esterne
- Notification system multi-canale
- License management
- Moduli attivabili/disattivabili singolarmente


## Configurazione

Vai su **Immens > Settings** per configurare:

1. **AI Providers**: aggiungi le tue API key per OpenAI, Claude, Gemini, ecc.
2. **Moduli**: attiva solo i moduli che ti servono
3. **Webhook**: configura webhook per eventi specifici
4. **Notifiche**: configura canali di notifica (email, Telegram)
