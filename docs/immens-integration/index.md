# Immens Integration

**Tipo**: PRO | **Versione**: 1.6.5 | WP: WordPress 5.8+ | PHP: PHP 7.4+
**Slug**: `immens-integration`

Layer di orchestrazione intelligente per WordPress. Gestisce AI providers (OpenAI, Claude, Gemini, OpenRouter, Perplexity), integrazioni con servizi terzi, moduli funzionali, e sistema di notifiche multi-canale.

## Caratteristiche

- AI Manager multi-provider: OpenAI, Claude, Gemini, OpenRouter, Perplexity, Custom OpenAI
- 18 integrazioni: Contact Form 7, Fluent Forms, Forminator, WooCommerce, CRM, Telegram, WhatsApp, SEO Framework, Primary Source, FluentSMTP
- 26 moduli funzionali: AdminNotepad, AI Image Generator, AutoArticle, Booking, Business Info, Calendar, Lead Magnet, QR Code, Uptime Monitor, Webhook, Newsletter AI
- Dashboard admin centralizzata
- Sistema di notifiche multi-canale
- Performance monitoring integrato
- Content sanitizer per AI
- Prompt templates predefiniti
