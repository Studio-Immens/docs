# Immens MCP Fortress

**Tipo**: FREE | **Versione**: 1.3.2 | WP: WordPress 6.0+ | PHP: PHP 7.4+
**Slug**: `immens-mcp-fortress`

Server MCP (Model Context Protocol) per WordPress. Connetti Claude, ChatGPT, Cursor, OpenClaw e qualsiasi AI compatibile MCP al tuo sito. Oltre 121 strumenti integrati per gestire ogni aspetto di WordPress via AI.

## Caratteristiche

- Server MCP streamable-http completo (JSON-RPC 2.0)
- 121 strumenti pre-integrati (posts, pages, media, comments, users, WooCommerce, SEO, traduzioni)
- Multi-Access-Point con API key indipendenti
- IP whitelist per access point (IPv4 + IPv6 CIDR)
- Rate limiting configurabile per access point
- OAuth 2.1 PKCE server integrato
- Audit logging completo di tutte le operazioni
- Auto-discovery endpoints per OpenClaw e OpenCode
- Session management con TTL 1 ora
- REST API gateway per qualsiasi namespace WordPress
- Supporto multisite (rete WordPress)
- CORS headers configurabili
