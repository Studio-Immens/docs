# Quick Order

**Tipo**: FREE | **Versione**: 1.0.0 | WP: WordPress 6.5+ | PHP: PHP 7.4+
**Dipende da**: WooCommerce
**Slug**: `studio-immens-quick-order`

Modulo di ordinazione rapida per WooCommerce pensato per ristoranti. Gestione tavoli, ordinazione via QR code, notifiche audio, e dashboard ordini in tempo reale.

## Caratteristiche

- Interfaccia ordinazione frontend AJAX, responsive mobile
- Gestione tavoli con assegnazione ordini (CPT siqo_tables)
- Generazione QR code per menu tavolo via shortcode
- 66+ suoni notifica per nuovi ordini
- Categorie personalizzate: ingredienti, allergeni, temperature, adesivi, macro-categorie, zone ristorante
- Stili personalizzabili via CSS variables
- Ruoli personalizzati: Worker, Supervisor, Manager
- Vista griglia/lista per gli ordini
- Catering management con data/ora consegna
- Restrizione GPS/rete locale
- Immagini per tassonomie (ingredienti, allergeni, etc.)
- Shortcode: [SIQO_front_order_ajax_section], [SIQO_manage_tables_section], [SIQO_manage_restaurant_section], [siqo_qr_code]
- Traduzione italiana inclusa
