# ⚡ Quick Order

**Tipo**: FREE
**Plugin**: studio-immens-quick-order

Aggiunge un modulo di ordine rapido per WooCommerce. Permette ai clienti di aggiungere prodotti al carrello inserendo direttamente il codice prodotto e la quantita'.

## Caratteristiche

- Modulo di ordine rapido via shortcode
- Ricerca prodotti in tempo reale
- Supporto variazioni prodotto
- Caricamento bulk da file CSV


## Utilizzo

Inserisci lo shortcode `[quick_order]` in qualsiasi pagina per mostrare il modulo di ordine rapido.

## Shortcode

```
[quick_order]
[quick_order show_search="true" show_bulk="false"]
```
