# Quick Order - Configurazione

## Shortcode

```
[quick_order]
[quick_order show_search="true" show_bulk="false"]
```

## Impostazioni

Vai su **WooCommerce > Quick Order** per configurare:

- **Mostra ricerca prodotti**: attiva/disattiva
- **Abilita carrello rapido**: aggiunge direttamente al carrello senza passare dalla pagina prodotto
- **Supporto variazioni**: permette di selezionare varianti prodotto
