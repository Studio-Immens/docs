# Quick Order - Configurazione

## Configurazione

### Shortcode

Inserisci gli shortcode nelle pagine del tuo sito:

| Shortcode | Descrizione |
|-----------|-------------|
| `[SIQO_front_order_ajax_section]` | Interfaccia ordinazione clienti |
| `[SIQO_manage_tables_section]` | Gestione tavoli |
| `[SIQO_manage_restaurant_section]` | Gestione ristorante |
| `[siqo_qr_code size="200"]` | QR code per la pagina corrente |

### Impostazioni

1. Vai su **Quick Order > Impostazioni**
2. Configura:
   - **Notifiche audio**: scegli tra 66+ suoni
   - **Timer aggiornamento**: frequenza refresh AJAX ordini
   - **Visualizzazione**: griglia/lista, colonne visibili
   - **Ruoli utente**: assegna permessi Worker/Supervisor/Manager

### Tassonomie

Il plugin registra tassonomie personalizzate per i prodotti:
- **Ingredienti** - Lista ingredienti per piatto
- **Allergeni** - Informazioni allergeni
- **Temperature** - Categorie temperatura (caldo/freddo)
- **Adesivi** - Etichette visive (novità, chef's special, etc.)
- **Macro-categorie** - Raggruppamenti menu
- **Zone** - Aree del ristorante (sala interna, esterno, etc.)
