# Immens CRM Core

**Tipo**: FREE
**Plugin**: immens-crm-core

Plugin CRM integrato in WordPress con gestione contatti, pipeline vendita, automazioni, email marketing, e analisi AI. Progettato per piccole e medie imprese.

## Caratteristiche

- Gestione contatti con campi personalizzati
- Pipeline di vendita con stage configurabili
- Automazioni e sequenze email
- Analisi AI delle performance vendita
- Tag e segmentazione contatti
- Integrazione WooCommerce
- Report e dashboard


## Configurazione

1. Vai su **CRM > Settings**
2. Configura le pipeline di vendita
3. Importa i contatti esistenti
4. Imposta le automazioni

## Tool MCP

Il CRM e' accessibile anche via MCP.

Esempi:
```
studioimmens__crm_list_contacts()
studioimmens__crm_get_contact(id=42)
studioimmens__crm_get_stats(range="30days")
```
