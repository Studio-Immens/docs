# Immens CRM Core - Installazione

## Installazione

1. Vai su **Plugin > Aggiungi nuovo > Carica plugin**
2. Carica il file ZIP
3. Attiva

## Primi Passi

1. Vai su **CRM > Settings**
2. Configura le pipeline di vendita
3. Importa i contatti esistenti
