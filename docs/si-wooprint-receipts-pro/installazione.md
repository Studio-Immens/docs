# WooPrint Receipts Pro - Installazione

## Installazione

1. Attiva license key
2. Installa WooPrint Receipts Pro
3. Configura su **WooCommerce > WooPrint Receipts**

## Requisiti

- WooPrint Receipts (free) installato
