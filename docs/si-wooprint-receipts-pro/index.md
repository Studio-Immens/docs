# WooPrint Receipts Pro

**Tipo**: PRO
**Plugin**: si-wooprint-receipts-pro

Versione Pro con supporto multi-stampante, stampa remota via PrintNode, PDF export, e integrazione con casse registratrici fiscali italiane (RT).

## Caratteristiche

- Multi-stampante (stampa contemporanea su piu' dispositivi)
- Stampa remota via PrintNode
- Export PDF delle ricevute
- Integrazione Registratore Telematico (RT)
- Template multipli per diversi tipi di ordine
- Filtri di stampa avanzati (per categoria, metodo pagamento, ecc.)


## Funzionalita' Pro

### PrintNode Integration
Stampa su stampanti remote via PrintNode senza necessita' di connessione diretta.

### Registratore Telematico (RT)
Integrazione con i registratori telematici italiani per la trasmissione dei corrispettivi all'Agenzia delle Entrate.

### PDF Export
Esporta qualsiasi ricevuta in PDF per inviarla via email al cliente.
