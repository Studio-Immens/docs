# WooPrint Receipts Pro

**Tipo**: PRO | **Versione**: 1.1.0 | WP: WordPress 5.8+ | PHP: PHP 7.4+
**Dipende da**: WooPrint Receipts (free) + WooCommerce 6.0+
**Slug**: `si-wooprint-receipts-pro`

Stampa ESC/POS diretta su stampanti termiche di rete, supporto USB via Print Node, regole avanzate per categoria/tag/prodotto, cassetto contanti, barcode CODE39, e licenza HMAC.

## Caratteristiche

- ESC/POS network printing diretto su stampanti termiche TCP/IP (porta 9100)
- Supporto Epson, Star e compatibili ESC/POS
- USB Print Node: app Node.js bridge per stampanti USB/seriali
- Regole avanzate: filtra per categoria prodotto, tag prodotto, prodotto specifico
- Cassetto contanti: apertura automatica via ESC/POS
- Barcode CODE39 su ricevuta con ID ordine
- Renderer immagini NV Graphics su stampanti termiche
- Set caratteri: CP437, CP850, CP858, UTF-8
- Multi-stampante illimitata (cucina, bar, magazzino...)
- Licenza HMAC SHA256 + aggiornamenti automatici
