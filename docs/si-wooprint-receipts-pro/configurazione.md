# WooPrint Receipts Pro - Configurazione

## Multi-Stampante

Collega piu' stampanti e stampa contemporaneamente su tutte.

## PrintNode

Stampa su stampanti remote senza connessione diretta:
1. Crea un account [PrintNode](https://www.printnode.com)
2. Installa il client PrintNode sulla macchina collegata alla stampante
3. Inserisci l'API key in WooPrint Receipts Pro

## Registratore Telematico (RT)

Integrazione con registratori telematici italiani per:
- Trasmissione corrispettivi all'Agenzia delle Entrate
- Generazione file RT
- Archivio digitale delle ricevute

## PDF Export

Esporta qualsiasi ricevuta in PDF per inviarla via email al cliente.
