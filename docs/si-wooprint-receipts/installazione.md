# WooPrint Receipts - Installazione

## Installazione

1. **Da WordPress**: Vai su **Plugin > Aggiungi nuovo** e cerca "WooPrint Receipts"
2. **Manuale**: Carica il file ZIP su **Plugin > Aggiungi nuovo > Carica plugin**
3. Attiva il plugin

### Requisiti

- WooCommerce 6.0+
