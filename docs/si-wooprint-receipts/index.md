# WooPrint Receipts

**Tipo**: FREE
**Plugin**: si-wooprint-receipts

Plugin per la stampa di ricevute e scontrini per WooCommerce. Supporta stampanti termiche ESC/POS, formati personalizzabili e stampa automatica alla ricezione dell'ordine.

## Caratteristiche

- Stampa ricevute per ordini WooCommerce
- Supporto stampanti termiche ESC/POS
- Template personalizzabili
- Stampa automatica alla ricezione ordine
- Formato scontrino o ricevuta A4
- Logo e intestazione personalizzabile


## Configurazione

1. Vai su **WooCommerce > WooPrint Receipts**
2. Configura la stampante (IP o porta USB)
3. Personalizza il template ricevuta
4. Scegli quando stampare: a ogni ordine, solo pagati, manualmente

## Shortcode

```
[wooprint_receipt order_id="123"]
```
