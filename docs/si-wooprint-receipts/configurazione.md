# WooPrint Receipts - Configurazione

## Configurazione Stampante

1. Vai su **WooCommerce > WooPrint Receipts**
2. Seleziona il tipo di stampante:
   - **ESC/POS termica** (IP o USB)
   - **A4** (stampa normale)
3. Personalizza il template

## Stampa Automatica

Puoi impostare la stampa automatica:
- A ogni nuovo ordine
- Solo ordini pagati
- Manualmente dal pannello ordini

## Shortcode

```
[wooprint_receipt order_id="123"]
```
