# ClassyBlocks - Installazione

## Installazione

1. Cerca "ClassyBlocks" su **Plugin > Aggiungi nuovo**
2. Installa e attiva

Oppure scarica da [WordPress.org](https://wordpress.org/plugins/studio-immens-css-classes/)

## Requisiti

- WordPress 6.0+
- Block Editor (Gutenberg)
