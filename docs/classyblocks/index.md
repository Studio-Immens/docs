# ClassyBlocks

**Tipo**: FREE | **Versione**: 2.3.2 | WP: WordPress 5.8+ | PHP: PHP 7.4+
**Slug**: `studio-immens-css-classes`

Aggiungi classi CSS personalizzate ai blocchi Gutenberg con live preview, supporto di 9 framework CSS, e un potente gestore di classi con esportazione/importazione.

## Caratteristiche

- 9 framework CSS integrati: Bootstrap 5.3.3, Tailwind CSS, Bulma, UIkit, Materialize, Pure CSS, Spectre, Semantic UI, Foundation
- Live preview delle classi nell'editor Gutenberg
- Editor CSS con CodeMirror e 4 pseudo-stati (:hover, :focus, :active, :visited)
- 5 preset pack precaricati (~140 classi): Layout, Spacing, Typography, Effects, Interactive
- Pack Manager: crea, esporta e importa pacchetti di classi
- Tailwind custom config via JSON
- Smart loading dei framework solo se necessari
- Blocco Gutenberg custom per CSS arbitrario per post
- Statistiche dashboard delle classi
- Traduzione italiana completa
- Supporto WP Consent API
