# 🎨 ClassyBlocks

**Tipo**: FREE
**Plugin**: classyblocks

Gestisci classi CSS personalizzate nei blocchi Gutenberg con live preview. Supporta Bootstrap, Tailwind CSS, Materialize, Bulma, UIkit, Spectre, PureCSS e Semantic.

## Caratteristiche

- Interfaccia visiva per gestire classi CSS
- 8 framework CSS integrati (Bootstrap, Tailwind, Materialize, Bulma, UIkit, Spectre, PureCSS, Semantic)
- Live preview nell'editor Gutenberg
- Tailwind CSS bundle locale (nessuna dipendenza CDN)
- Zero impatto sulle performance del frontend
- CSS sanitization integrata


## Installazione

1. Vai su **Plugin > Aggiungi nuovo** in WordPress
2. Cerca "ClassyBlocks"
3. Installa e attiva

## Configurazione

Vai nella sezione **CSS Classes** nel menu di WordPress. Qui puoi:

1. **Creare classi CSS personalizzate** con nome e proprieta'
2. **Scegliere un framework CSS** da abilitare
3. **Applicare le classi ai blocchi** direttamente dalla sidebar dell'editor Gutenberg
