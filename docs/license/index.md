# 🔑 Studio Immens License

**Tipo**: PRO
**Plugin**: license

Sistema di licenze per i plugin Pro di Studio Immens. Gestisce attivazione, validazione e rinnovo delle licenze dei plugin.

## Caratteristiche

- Attivazione licenza via API
- Validazione periodica automatica
- Supporto multi-sito
- Gestione rinnovi e scadenze
- Deactivate su cambio dominio


## Attivazione

1. Vai su **Immens > License** nel menu WordPress
2. Inserisci la tua license key
3. Clicca "Attiva"
4. La licenza viene validata automaticamente ogni 24 ore

Se la licenza scade, i plugin Pro continuano a funzionare ma non ricevono aggiornamenti.
