# Primary Source

**Tipo**: PRO | **Versione**: 1.0.0 | WP: WordPress 5.8+ | PHP: PHP 7.4+
**Slug**: `primary-source`

Definisce la "sorgente primaria" per agenti Paperclip/OpenClaw. Determina se le istruzioni dell'agente provengono dalla directory gestita da Paperclip o da un workspace esterno con AGENTS.md, SOUL.md, TOOLS.md, ecc.

## Caratteristiche

- Due modalità: managed (Paperclip gestisce) o external (workspace locale)
- Configurazione instructionsRootPath per workspace esterno
- Entry point personalizzabile (default: AGENTS.md)
- Supporto per file AGENTS.md, SOUL.md, TOOLS.md, HEARTBEAT.md, MEMORY.md, USER.md
- Integrazione nativa con Paperclip gateway adapter
