# MCP Fortress Pro - Installazione

## Installazione

1. Attiva la **license key** su **Immens > License**
2. Installa il plugin Pro (sovrascrive la versione free)
3. Verifica su **MCP Fortress > Dashboard**

## Requisiti

- Immens MCP Fortress (free) installato
- License key valida
