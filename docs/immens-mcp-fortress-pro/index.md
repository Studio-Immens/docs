# 🛡️ Immens MCP Fortress Pro

**Tipo**: PRO
**Plugin**: immens-mcp-fortress-pro

La versione Pro aggiunge funzionalita' avanzate per team e agenzie: approval workflow con Telegram, supporto OAuth, access points illimitati, e audit log avanzato.

## Caratteristiche

- Approvals via Telegram per operazioni sensibili
- Access Points illimitati
- OAuth integration
- Audit log avanzato con retention configurabile
- Rate limiting per access point
- Supporto prioritario


## Funzionalita' Pro

### Approval Workflow
Le operazioni di scrittura (creare post, aggiornare prodotti, eliminare contenuti) richiedono approvazione via Telegram prima di essere eseguite.

### OAuth
Supporto per autenticazione OAuth 2.0 per integrazioni con servizi esterni.

### Access Points Illimitati
Crea quanti access points vuoi, ognuno con permessi e IP whitelist diversi. Ideale per agenzie che gestiscono decine di siti.
