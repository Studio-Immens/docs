# Fast Product Importer - Installazione

## Installazione

1. **Da WordPress**: Vai su **Plugin > Aggiungi nuovo > Carica plugin**
2. Carica il file ZIP
3. Attiva il plugin

### Requisiti

- WooCommerce 5.0+
- (Opzionale) API key per AI: OpenAI, Claude, Gemini, OpenRouter
