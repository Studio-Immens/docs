# Fast Product Importer

**Tipo**: FREE | **Versione**: 1.2.1 | WP: WordPress 5.8+ | PHP: PHP 7.4+
**Dipende da**: WooCommerce 5.0+
**Slug**: `si-fast-product-importer`

Importa prodotti in WooCommerce da un vasto database remoto con generazione AI di descrizioni, excerpt e SEO meta. Supporta OpenAI, Claude, Gemini e OpenRouter.

## Caratteristiche

- Ricerca istantanea prodotti da database remoto
- Importazione一键 con immagini, prezzi e SEO meta
- Generatore AI descrizioni (OpenAI, Claude, Gemini, OpenRouter)
- Database demo locale: 2000 prodotti pre-generati (IT/EN)
- Campi personalizzabili: SKU prefix, stock, stato prodotto
- Supporto prodotti variabili con attributi e variazioni
- Meta SEO per Yoast SEO e Rank Math
- Caricamento immagini main + galleria da URL
- Creatività AI regolabile (temperature)
- API key crittografate nel database
- Cache risultati ricerca (1 ora)
- Log errori integrato (ultimi 50)
