# Quick Order Pro

**Tipo**: PRO | **Versione**: 1.0.0 | WP: WordPress 6.5+ | PHP: PHP 7.4+
**Dipende da**: Quick Order (free) + WooCommerce
**Slug**: `studio-immens-quick-order-pro`

Estensione premium per Quick Order con gestione multi-area, statistiche real-time, REST API completa, timer ordini configurabile e gestione prioritaria.

## Caratteristiche

- Multi-area table management (zone ristorante)
- Statistiche ordini in tempo reale e revenue
- REST API completa per integrazioni esterne
- Timer ordini configurabile (durata + scadenza)
- Refresh rate AJAX personalizzabile per tavoli
- Priority order management
- Statistiche dashboard
