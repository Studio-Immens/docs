# SIWP Reseller

**Tipo**: PRO
**Plugin**: siwp-reseller

Sistema di rivendita white-label per i plugin Studio Immens. Permette di gestire clienti, assegnare licenze, e fatturare automaticamente.

## Caratteristiche

- Gestione clienti rivenditori
- Assegnazione e revoca licenze plugin
- Fatturazione automatica
- Pannello white-label per i tuoi clienti
- API per integrazione con sistemi esterni


## Configurazione

1. Vai su **SIWP Reseller > Dashboard**
2. Aggiungi i tuoi clienti
3. Assegna licenze plugin
4. I clienti ricevono email con le credenziali di accesso

## API

```
GET /wp-json/siwp-reseller/v1/licenses
POST /wp-json/siwp-reseller/v1/licenses/assign
```
