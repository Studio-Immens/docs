# SIWP Reseller

**Tipo**: PRO | **Versione**: 1.0.1 | WP: WordPress 5.8+ | PHP: PHP 7.4+
**Dipende da**: WooCommerce 5.0+
**Slug**: `siwp-reseller`

Sistema di rivendita white-label per plugin Studio Immens. Gestione clienti, fatture, preventivi, DDT, magazzino, licenze e fatturazione elettronica (FatturaPA SDI).

## Caratteristiche

- CRM rivenditori con profili dettagliati e pipeline
- Preventivi con tre livelli di prezzo (standard, base, premium)
- Fatture con numerazione sezionale e FatturaPA XML
- DDT (Documento di Trasporto) con tracking carrier
- Magazzino con movimenti carico/scarico e previsioni stock
- Integrazione WooCommerce profonda (ordini avanzati, pricing tier)
- Generazione PDF, CSV, DOCX per documenti
- FatturaPA SDI con invio e callback di stato
- Ticket di supporto collegati a ordini/fatture
- Portale rivenditori self-service con shortcode [siwp_reseller_dashboard]
- Licenza HMAC SHA256 con aggiornamenti automatici
- PSR-4 autoloading, 18 controller, 6 servizi
- 80+ AJAX endpoint per tutte le operazioni
