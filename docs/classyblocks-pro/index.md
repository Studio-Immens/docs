# ClassyBlocks Pro

**Tipo**: PRO | **Versione**: 1.3.6 | WP: WordPress 5.8+ | PHP: PHP 7.4+
**Dipende da**: ClassyBlocks (free)
**Slug**: `classyblocks-pro`

Animazioni scroll-driven, effetti hover, text splitting, cursor tilt 3D e molto altro. Oltre 20 animazioni CSS pure in 12 preset pack, con licenza HMAC e aggiornamenti automatici.

## Caratteristiche

- Oltre 20 animazioni scroll-driven pure CSS (animation-timeline: view())
- 12 preset pack: Essential Scroll, Hero Effects, Attention Grabbers, Background, Entrance, Hover, Loading, Mask Reveal, Mouse Magic, Special Scroll, Text Effects
- Categorie: scroll, attention, text, hover, entrance, reveal, cursor
- Anteprima live animazioni nell'editor Gutenberg
- Override per-blocco: durata, easing, delay, iterazioni, direzione
- Animazioni text-split (fade letters, wave, pop)
- Effetto cursore 3D tilt (cb-cursor-tilt-3d)
- IntersectionObserver per animazioni di entrata
- Polyfill JS per Firefox/Safari (scroll-driven animation fallback)
- Pack Manager per animazioni con import/export JSON
- Licenza HMAC SHA256 con attivazione remota
- Aggiornamenti automatici via license server
- Backup automatico su disattivazione
