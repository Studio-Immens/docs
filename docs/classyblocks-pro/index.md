# 🎨 ClassyBlocks Pro

**Tipo**: PRO
**Plugin**: classyblocks-pro

La versione Pro aggiunge animazioni CSS, classi condizionali, template di classi riutilizzabili e integrazione con page builder.

## Caratteristiche

- Animazioni CSS avanzate (scroll reveal, fade, slide)
- Classi condizionali (solo per utenti loggati, solo mobile, ecc.)
- Template di classi riutilizzabili
- Integrazione con Elementor e WPBakery
- Esportazione/importazione configurazioni


## Funzionalita' Pro

### Animazioni
Aggiungi animazioni scroll-reveal, fade-in, slide-up e molte altre direttamente dai blocchi Gutenberg.

### Classi Condizionali
Mostra classi CSS solo in determinate condizioni: utente loggato, dispositivo mobile, pagina specifica, ecc.

### Template
Crea combinazioni di classi riutilizzabili e applicale con un click.
