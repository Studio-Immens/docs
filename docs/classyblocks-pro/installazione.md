# ClassyBlocks Pro - Installazione

## Installazione

1. **Attiva la licenza**: Vai su **Immens > License** e inserisci la license key
2. **Installa il plugin**: Carica il file ZIP su **Plugin > Aggiungi nuovo > Carica plugin**
3. ClassyBlocks Pro si integra automaticamente con ClassyBlocks (free)

### Requisiti

- ClassyBlocks (free) installato e attivo
- License key valida
