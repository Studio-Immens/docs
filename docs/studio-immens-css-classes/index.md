# 🎯 Studio Immens CSS Classes

**Tipo**: FREE
**Plugin**: studio-immens-css-classes

Plugin WordPress per aggiungere classi CSS personalizzate a qualsiasi blocco Gutenberg. Ideale per sviluppatori che vogliono controllo totale sull'aspetto dei blocchi.

## Caratteristiche

- Aggiunge campo "CSS Class" a tutti i blocchi Gutenberg
- Supporto classi multiple
- Autocomplete con classi utilizzate di recente
- Leggero: nessun caricamento extra sul frontend


## Utilizzo

1. Installa e attiva il plugin
2. Apri un qualsiasi post/pagina con Gutenberg
3. Seleziona un blocco
4. Nella sidebar, trova il campo "CSS Class"
5. Inserisci le classi desiderate (separate da spazio)
