# Immens Keyword Explorer (IKE) - Installazione

## Installazione

1. **Attiva la licenza**: Vai su **Immens > License** e inserisci la license key
2. Carica il file ZIP su **Plugin > Aggiungi nuovo > Carica plugin**
3. Attiva il plugin

### Requisiti

- Immens Integration (plugin core)
- License key valida
